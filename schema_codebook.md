Park Facilities Quality Tracker – Schema Codebook

Table: quality_reports

This table stores reports about park facility issues submitted through the Shiny application.

id Unique identifier for each report.

park_name Name of the park where the issue was reported.

issue_type Type of issue reported (for example broken equipment or unsafe surface).

severity Severity level of the issue from 1 to 5.

notes Optional description or additional details about the issue.

reported_by Name of the person who submitted the report.

reported_at Timestamp automatically generated when the report is created.
